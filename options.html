Usage

  • Config must be array of rules separated by newline
  • Each rule is object of two arrays: origins, domains separated by "-"
  • both "origins" and "domains" are arrays of url patterns described at Match Patterns page separated by newline
  • example: 
    *://origin.example.com/* origin url
*://additional.example.com/* additional origin url
- origin and domains urls are separated by "-"
*://target.example.org/* target domain url
*://additional.example.org/* additional target domain url
rules are separated by double newline
*://mozilla.org/*
*://*.mozilla.org/*
-
*://example.org/*
*://*.example.org/*
